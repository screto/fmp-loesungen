Aufgabe 3.22 – Potenzgesetze

Nächste Aufgabe »

Vereinfachen Sie:

  1. a)$\frac{15^4}{5^4} =\ ?$ $\frac{15^4}{5^4} = \left(\frac{15}{5}\right)^4 = 3^4 = 81$
  2. b)$\frac{0.75^4}{3^4} =\ ?$ $\frac{0.75^4}{3^4} = \left(\frac{0.75}{3}\right)^4 = 0.25^4 = \frac{1}{256}$
  3. c)$\frac{0.25^3}{0.5^3} =\ ?$ $\frac{0.25^3}{0.5^3} = \left(\frac{0.25}{0.5}\right)^3 = 0.5^3 = \frac{1}{8}$
  4. d)$\frac{a^6}{b^6} =\ ?$ $\frac{a^6}{b^6} = \left(\frac{a}{b}\right)^6$
  5. e)$\frac{(a^2-b^2)^4}{(a-b)^4} =\ ?$ $\frac{(a^2-b^2)^4}{(a-b)^4} = \left(\frac{a^2-b^2}{a-b}\right)^4 = (a+b)^4$
  6. f)$\frac{x^6}{(-y)^6} =\ ?$ $\frac{x^6}{(-y)^6} = \left(\frac{x}{-y}\right)^6 = \left(-\frac{x}{y}\right)^6 = \left(\frac{x}{y}\right)^6$
  7. g)$\frac{a^{(m-2)}}{b^{(m-2)}} =\ ?$ $\frac{a^{(m-2)}}{b^{(m-2)}} = \left(\frac{a}{b}\right)^{m-2}$
  8. h)$\frac{(a^2-b^2)^{2m}}{(a-b)^{2m}} =\ ?$ $\frac{(a^2-b^2)^{2m}}{(a-b)^{2m}} = \left(\frac{a^2-b^2}{a-b}\right)^{2m} = (a+b)^{2m}$
  9. i)$\frac{(2a)^n}{a^n} =\ ?$ $\frac{(2a)^n}{a^n} = \left(\frac{2a}{a}\right)^n = 2^n$
  10. j)$\frac{(ab)^n}{a^n} =\ ?$ $\frac{(ab)^n}{a^n} = \left(\frac{ab}{a}\right)^n = b^n$
  11. k)$\frac{(12)^n}{(18x)^n} =\ ?$ $\frac{(12)^n}{(18x)^n} = \left(\frac{12}{18x}\right)^n = \left(\frac{2}{3x}\right)^n$
  12. l)$\frac{(3a^2+2a)^x}{a^x} =\ ?$ $\frac{(3a^2+2a)^x}{a^x} = \left(\frac{3a^2+2a}{a}\right)^x = (3a+2)^x$
