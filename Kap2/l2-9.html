Lösen Sie nach der jeweils angegebenen Variablen auf:

Aufgabe Lösung
1) $\dfrac{a + b}{c} = \dfrac{d}{e - f} \;\Rightarrow\; b = \ ?$
$b = \dfrac{c\,d}{\,e - f\,} - a$
2) $\dfrac{a + b}{c} = \dfrac{d}{e - f} \;\Rightarrow\; e = \ ?$
$e = f + \dfrac{c\,d}{\,a + b\,}$
3) $\dfrac{a + b}{c} = \dfrac{d}{e - f} \;\Rightarrow\; f = \ ?$
$f = e - \dfrac{c\,d}{\,a + b\,}$
4) $\dfrac{1}{a} = \dfrac{1}{b} + \dfrac{1}{c} \;\Rightarrow\; a = \ ?$
$a = \dfrac{b\,c}{\,b + c\,}$
5) $\dfrac{1}{a} = \dfrac{1}{b} + \dfrac{1}{c} \;\Rightarrow\; b = \ ?$
$b = \dfrac{a\,c}{\,c - a\,}$
6) $\dfrac{1}{a} = \dfrac{1}{b} + \dfrac{1}{c} \;\Rightarrow\; c = \ ?$
$c = \dfrac{a\,b}{\,b - a\,}$
7) $\dfrac{1}{a} = \dfrac{1}{b - c} + \dfrac{1}{d} \;\Rightarrow\; a = \ ?$
$a = \dfrac{d\,(b - c)}{\,b - c + d\,}$
8) $\dfrac{1}{a} = \dfrac{1}{b - c} + \dfrac{1}{d} \;\Rightarrow\; c = \ ?$
$c = b + \dfrac{a\,d}{\,a - d\,}$
9) $\dfrac{1}{a} = \dfrac{1}{b - c} + \dfrac{1}{d} \;\Rightarrow\; d = \ ?$
$d = \dfrac{a\,(b - c)}{\,b - c - a\,}$
10) $\dfrac{a}{b + c} = \dfrac{d - e}{f} \;\Rightarrow\; e = \ ?$
$e = d - \dfrac{f\,a}{\,b + c\,}$
