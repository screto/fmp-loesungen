Aufgabe 3.40 - Bruchgleichungen lösen

Aufgabe 3.41 »

Aufgabe: Lösen Sie die nachfolgenden Bruchgleichungen. Beachten Sie zwingend die Definitionsmenge. Als Grundmenge gilt bei allen Aufgaben $\mathbb{G}=\mathbb{R}$.

a) $\frac{1}{x} - \frac{1}{3 \cdot x} = \frac{8}{9}$

$\mathbb{D} = \mathbb{R} \setminus \{0\}$

$\frac{1}{x} - \frac{1}{3x} = \frac{8}{9}$

$\frac{3 - 1}{3x} = \frac{8}{9}$

$\frac{2}{3x} = \frac{8}{9}$

$2 \cdot 9 = 8 \cdot 3x$

$18 = 24x$

$x = \frac{18}{24} = \frac{3}{4}$

$\mathbb{L} = \{\frac{3}{4}\}$

b) $\frac{2}{x} = 1 + \frac{2}{x}$

$\mathbb{D} = \mathbb{R} \setminus \{0\}$

$\frac{2}{x} - \frac{2}{x} = 1$

$0 = 1$

Dies ist ein Widerspruch.

$\mathbb{L} = \{\}$

c) $2 = \frac{7}{x - 3}$

$\mathbb{D} = \mathbb{R} \setminus \{3\}$

$2(x - 3) = 7$

$2x - 6 = 7$

$2x = 13$

$x = \frac{13}{2}$

$\mathbb{L} = \{\frac{13}{2}\}$

d) $\frac{-3}{2 - x} = \frac{3}{x - 2}$

$\mathbb{D} = \mathbb{R} \setminus \{2\}$

Da $2 - x = -(x - 2)$:

$\frac{-3}{-(x-2)} = \frac{3}{x - 2}$

$\frac{3}{x-2} = \frac{3}{x - 2}$

Die Gleichung ist für alle $x \in \mathbb{D}$ erfüllt.

$\mathbb{L} = \mathbb{R} \setminus \{2\}$

e) $\frac{1}{\frac{3}{x} + 1} = 1 - \frac{2}{x+3}$

$\mathbb{D} = \mathbb{R} \setminus \{-3; 0\}$

$\frac{1}{\frac{3+x}{x}} = 1 - \frac{2}{x+3}$

$\frac{x}{3+x} = 1 - \frac{2}{x+3}$

$\frac{x}{x+3} = \frac{x+3-2}{x+3}$

$\frac{x}{x+3} = \frac{x+1}{x+3}$

$x = x + 1$

$0 = 1$

Dies ist ein Widerspruch.

$\mathbb{L} = \{\}$

f) $\frac{4x + 1}{4x + 2} - \frac{5x - 2}{6x + 3} = \frac{5}{18}$

$\mathbb{D} = \mathbb{R} \setminus \{-\frac{1}{2}\}$

$\frac{4x + 1}{2(2x + 1)} - \frac{5x - 2}{3(2x + 1)} = \frac{5}{18}$

$\frac{3(4x + 1) - 2(5x - 2)}{6(2x + 1)} = \frac{5}{18}$

$\frac{12x + 3 - 10x + 4}{6(2x + 1)} = \frac{5}{18}$

$\frac{2x + 7}{6(2x + 1)} = \frac{5}{18}$

$18(2x + 7) = 30(2x + 1)$

$36x + 126 = 60x + 30$

$96 = 24x$

$x = 4$

$\mathbb{L} = \{4\}$

g) $\frac{5}{3 - x} = \frac{-5}{x - 3}$

$\mathbb{D} = \mathbb{R} \setminus \{3\}$

Da $3 - x = -(x - 3)$:

$\frac{5}{-(x-3)} = \frac{-5}{x - 3}$

$\frac{-5}{x-3} = \frac{-5}{x - 3}$

Die Gleichung ist für alle $x \in \mathbb{D}$ erfüllt.

$\mathbb{L} = \mathbb{R} \setminus \{3\}$

h) $\frac{x^2 + 3}{4x^2 + 12x + 9} = \frac{2x + 1}{4x+6} - \frac{x+1}{4x}$

$4x^2 + 12x + 9 = (2x + 3)^2$

$\mathbb{D} = \mathbb{R} \setminus \{-\frac{3}{2}; 0\}$

$\frac{x^2 + 3}{(2x + 3)^2} = \frac{2x + 1}{2(2x+3)} - \frac{x+1}{4x}$

Hauptnenner: $4x(2x+3)^2$

$\frac{4x(x^2 + 3)}{4x(2x + 3)^2} = \frac{2x(2x+3)(2x + 1) - (x+1)(2x+3)^2}{4x(2x+3)^2}$

$4x(x^2 + 3) = 2x(2x+3)(2x + 1) - (x+1)(2x+3)^2$

$4x^3 + 12x = (2x+3)[2x(2x+1) - (x+1)(2x+3)]$

$4x^3 + 12x = (2x+3)[4x^2 + 2x - 2x^2 - 3x - 2x - 3]$

$4x^3 + 12x = (2x+3)(2x^2 - 3x - 3)$

$4x^3 + 12x = 4x^3 - 6x^2 - 6x + 6x^2 - 9x - 9$

$4x^3 + 12x = 4x^3 - 15x - 9$

$27x = -9$

$x = -\frac{1}{3}$

$\mathbb{L} = \{-\frac{1}{3}\}$

i) $\frac{2x - 1}{x + 2} - \frac{x + 2}{2x-1} = \frac{8}{3}$

$\mathbb{D} = \mathbb{R} \setminus \{-2; \frac{1}{2}\}$

Mit $(x+2)(2x-1)$ multiplizieren:

$(2x-1)^2 - (x+2)^2 = \frac{8}{3}(x+2)(2x-1)$

$4x^2 - 4x + 1 - x^2 - 4x - 4 = \frac{8}{3}(2x^2 + 3x - 2)$

$3x^2 - 8x - 3 = \frac{8}{3}(2x^2 + 3x - 2)$

$9x^2 - 24x - 9 = 16x^2 + 24x - 16$

$-7x^2 - 48x + 7 = 0$

$7x^2 + 48x - 7 = 0$

$x = \frac{-48 \pm \sqrt{2304 + 196}}{14} = \frac{-48 \pm \sqrt{2500}}{14} = \frac{-48 \pm 50}{14}$

$x_1 = \frac{2}{14} = \frac{1}{7}$, $x_2 = \frac{-98}{14} = -7$

$\mathbb{L} = \{-7; \frac{1}{7}\}$

j) $\frac{1}{x + 1} + \frac{1}{x - 1} = \frac{x^2 + 1}{x^2 - 1}$

$\mathbb{D} = \mathbb{R} \setminus \{-1; 1\}$

$\frac{1}{x + 1} + \frac{1}{x - 1} = \frac{x^2 + 1}{(x + 1)(x - 1)}$

$\frac{x - 1 + x + 1}{(x + 1)(x - 1)} = \frac{x^2 + 1}{(x + 1)(x - 1)}$

$\frac{2x}{(x + 1)(x - 1)} = \frac{x^2 + 1}{(x + 1)(x - 1)}$

$2x = x^2 + 1$

$x^2 - 2x + 1 = 0$

$(x - 1)^2 = 0$

$x = 1$, aber $1 \notin \mathbb{D}$

$\mathbb{L} = \{\}$

k) $\frac{x - 5}{x^2 - 5x} = \frac{x + 8}{x - 5}$

$x^2 - 5x = x(x - 5)$

$\mathbb{D} = \mathbb{R} \setminus \{0; 5\}$

$\frac{x - 5}{x(x - 5)} = \frac{x + 8}{x - 5}$

$\frac{1}{x} = \frac{x + 8}{x - 5}$

$(x - 5) = x(x + 8)$

$x - 5 = x^2 + 8x$

$x^2 + 7x + 5 = 0$

$x = \frac{-7 \pm \sqrt{49 - 20}}{2} = \frac{-7 \pm \sqrt{29}}{2}$

$x_1 = \frac{-7 + \sqrt{29}}{2} \approx -0.81$

$x_2 = \frac{-7 - \sqrt{29}}{2} \approx -6.19$

$\mathbb{L} = \{\frac{-7 - \sqrt{29}}{2}; \frac{-7 + \sqrt{29}}{2}\}$
