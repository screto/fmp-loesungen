Welche der folgenden Zuordnungen sind Funktionen? Begründen Sie bzw. widerlegen Sie!
a) Person $\to$ Körpergrösse
Funktion. Jede Person hat zu einem bestimmten Zeitpunkt genau eine eindeutige Körpergrösse.
b) Körpergrösse $\to$ Person
Keine Funktion. Einer Körpergrösse (z.B. 180 cm) können mehrere Personen zugeordnet sein.
c) Tag $\to$ Temperatur
Keine Funktion. An einem Tag gibt es viele verschiedene Temperaturen (Verlauf über 24h). Die Zuordnung ist nicht eindeutig.
d) Tag $\to$ Maximaltemperatur
Funktion. Pro Tag gibt es genau einen Wert, der das Maximum darstellt.
e) Temperatur $\to$ Tag
Keine Funktion. Eine bestimmte Temperatur (z.B. 20°C) kann an vielen verschiedenen Tagen auftreten.
f) Schulfach $\to$ Zeugnisnote
Funktion. Jedem Fach wird in einem Zeugnis genau eine Endnote zugeordnet.
g) Zeugnisnote $\to$ Schulfach
Keine Funktion. Man kann die Note 5.0 in mehreren verschiedenen Fächern haben.
