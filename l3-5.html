Gegeben sind die drei Mengen:
$A = \{1; 2; 3; 5; 6\}$
$B = \{2; 5; 6; 7; 8\}$
$C = \{5; 6; 9; 10\}$

  • a) $A \cup B$ $A \cup B = \{1; 2; 3; 5; 6; 7; 8\}$
  • b) $B \cap C$ $B \cap C = \{5; 6\}$
  • c) $(A \cup B) \cap (B \cup C)$ $(A \cup B) \cap (B \cup C) = \{2; 5; 6; 7; 8\}$
  • d) $(A \cap B) \cup (B \cap C)$ $(A \cap B) \cup (B \cap C) = \{2; 5; 6\}$
  • e) $(A \cup B) \cup (B \cap C)$ $(A \cup B) \cup (B \cap C) = \{1; 2; 3; 5; 6; 7; 8\}$
  • f) $A \setminus B$ $A \setminus B = \{1; 3\}$
  • g) $(A \setminus C) \cup (B \setminus C)$ $(A \setminus C) \cup (B \setminus C) = \{1; 2; 3; 7; 8\}$
  • h) $A \cap (B \cup C)$ $A \cap (B \cup C) = \{2; 5; 6\}$
  • i) $A \cup (B \cap C)$ $A \cup (B \cap C) = \{1; 2; 3; 5; 6\}$
  • j) $(A \cup B) \setminus C$ $(A \cup B) \setminus C = \{1; 2; 3; 7; 8\}$
