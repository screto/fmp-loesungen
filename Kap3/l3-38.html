Aufgabe 3.38 - Gleichungen mit verschiedenen Grundmengen

Aufgabe 3.39 »
Aufgabe: Lösen Sie die nachfolgenden Gleichungen. Beachten Sie die jeweilige Grundmenge:
a) $x - 4a - (x + 2b - (x - 6c)) = x - (5a - (x - 4b) + 9c)$
$\mathbb{G}=\mathbb{R}$
Linke Seite vereinfachen:
$x - 4a - (x + 2b - (x - 6c))$
$= x - 4a - (x + 2b - x + 6c)$
$= x - 4a - (2b + 6c)$
$= x - 4a - 2b - 6c$

Rechte Seite vereinfachen:
$x - (5a - (x - 4b) + 9c)$
$= x - (5a - x + 4b + 9c)$
$= x - 5a + x - 4b - 9c$
$= 2x - 5a - 4b - 9c$

Gleichung lösen:
$x - 4a - 2b - 6c = 2x - 5a - 4b - 9c$
$-4a - 2b - 6c + 5a + 4b + 9c = 2x - x$
$a + 2b + 3c = x$

Lösungsmenge: $\mathbb{L} = \{a + 2b + 3c\}$
b) $(5x + 1)^2 - (3x - 2)^2 = (4x + 3)^2 - 3 \cdot (x + 5)$
$\mathbb{G}=\mathbb{N}$
Linke Seite ausmultiplizieren:
$(5x + 1)^2 - (3x - 2)^2$
$= 25x^2 + 10x + 1 - (9x^2 - 12x + 4)$
$= 25x^2 + 10x + 1 - 9x^2 + 12x - 4$
$= 16x^2 + 22x - 3$

Rechte Seite ausmultiplizieren:
$(4x + 3)^2 - 3(x + 5)$
$= 16x^2 + 24x + 9 - 3x - 15$
$= 16x^2 + 21x - 6$

Gleichung lösen:
$16x^2 + 22x - 3 = 16x^2 + 21x - 6$
$22x - 3 = 21x - 6$
$x = -3$

Da $-3 \notin \mathbb{N}$: Lösungsmenge: $\mathbb{L} = \emptyset$
c) $(x + 2)^2 + 8 - (x + 2) \cdot (x - 4) = (x - 6)^2 - (x - 4)^2$
$\mathbb{G}=\mathbb{Q}$
Linke Seite ausmultiplizieren:
$(x + 2)^2 + 8 - (x + 2)(x - 4)$
$= x^2 + 4x + 4 + 8 - (x^2 - 4x + 2x - 8)$
$= x^2 + 4x + 12 - x^2 + 2x + 8$
$= 6x + 20$

Rechte Seite ausmultiplizieren:
$(x - 6)^2 - (x - 4)^2$
$= x^2 - 12x + 36 - (x^2 - 8x + 16)$
$= x^2 - 12x + 36 - x^2 + 8x - 16$
$= -4x + 20$

Gleichung lösen:
$6x + 20 = -4x + 20$
$10x = 0$
$x = 0$

Da $0 \in \mathbb{Q}$: Lösungsmenge: $\mathbb{L} = \{0\}$
d) $x^2 + (2a - x)(2a + x) - (x - b)^2 = 2a(a - x) - 4ab - (x + b)^2$
$\mathbb{G}=\mathbb{Q}$
Linke Seite vereinfachen:
$x^2 + (2a - x)(2a + x) - (x - b)^2$
$= x^2 + (4a^2 - x^2) - (x^2 - 2bx + b^2)$
$= x^2 + 4a^2 - x^2 - x^2 + 2bx - b^2$
$= 4a^2 - x^2 + 2bx - b^2$

Rechte Seite vereinfachen:
$2a(a - x) - 4ab - (x + b)^2$
$= 2a^2 - 2ax - 4ab - (x^2 + 2bx + b^2)$
$= 2a^2 - 2ax - 4ab - x^2 - 2bx - b^2$

Gleichung lösen:
$4a^2 - x^2 + 2bx - b^2 = 2a^2 - 2ax - 4ab - x^2 - 2bx - b^2$
$4a^2 + 2bx = 2a^2 - 2ax - 4ab - 2bx$
$2a^2 + 4bx + 2ax + 4ab = 0$
$2a^2 + 2ax + 4bx + 4ab = 0$
$2a(a + x) + 4b(x + a) = 0$
$(a + x)(2a + 4b) = 0$
$2(a + x)(a + 2b) = 0$

Da $a, b$ Parameter sind: $x = -a$

Lösungsmenge: $\mathbb{L} = \{-a\}$ für alle $a \in \mathbb{Q}$
