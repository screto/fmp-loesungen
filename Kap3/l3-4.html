Ergänzen Sie jeweils die fehlende Spalte (aufzählende oder beschreibende Form):

aufzählende Form $\Leftrightarrow$ beschreibende Form
a) $\{1; 2; 3; 4\}$ $\Leftrightarrow$
$\{x \mid x \in \mathbb{N} \text{ und } x < 5\}$
b)
$\{4; 5; 6\}$
$\Leftrightarrow$ $\{x \mid x \in \mathbb{N} \text{ und } 3 < x < 7\}$
c) $\{-5; -4; -3; -2\}$ $\Leftrightarrow$
$\{x \mid x \in \mathbb{Z} \text{ und } -6 < x < -1\}$
d)
$\{-1; 0; 1; 2\}$
$\Leftrightarrow$ $\{x \mid x \in \mathbb{Z} \text{ und } -2 < x < 3\}$
