Aufgabe 3.39 - Quadratische Gleichungen lösen

Aufgabe 3.40 »

Aufgabe: Lösen Sie die nachfolgenden Gleichungen. Beachten Sie die jeweilige Grundmenge:

a) $x^2 + 5x - 14 = 0$     $\mathbb{G}=\mathbb{R}$

Mit der Mitternachtsformel: $x = \frac{-5 \pm \sqrt{25 + 56}}{2} = \frac{-5 \pm \sqrt{81}}{2} = \frac{-5 \pm 9}{2}$

$x_1 = 2$, $x_2 = -7$

$\mathbb{L} = \{-7; 2\}$

b) $2x^2 + 8x + 6 = 0$     $\mathbb{G}=\mathbb{R}$

Durch 2 teilen: $x^2 + 4x + 3 = 0$

Mit der Mitternachtsformel: $x = \frac{-4 \pm \sqrt{16 - 12}}{2} = \frac{-4 \pm 2}{2}$

$x_1 = -1$, $x_2 = -3$

$\mathbb{L} = \{-3; -1\}$

c) $3x^2 - 36x + 105 = 0$     $\mathbb{G}=\mathbb{R}$

Durch 3 teilen: $x^2 - 12x + 35 = 0$

Mit der Mitternachtsformel: $x = \frac{12 \pm \sqrt{144 - 140}}{2} = \frac{12 \pm 2}{2}$

$x_1 = 7$, $x_2 = 5$

$\mathbb{L} = \{5; 7\}$

d) $(7 - x)(3x + 1) - (x - 5)(17 - 2x) = 84$     $\mathbb{G}=\mathbb{R}$

Ausmultiplizieren: $21x + 7 - 3x^2 - x - (17x - 2x^2 - 85 + 10x) = 84$

$20x + 7 - 3x^2 - 17x + 2x^2 + 85 - 10x = 84$

$-x^2 - 7x + 92 = 84$

$-x^2 - 7x + 8 = 0$ bzw. $x^2 + 7x - 8 = 0$

$x = \frac{-7 \pm \sqrt{49 + 32}}{2} = \frac{-7 \pm 9}{2}$

$x_1 = 1$, $x_2 = -8$

$\mathbb{L} = \{-8; 1\}$

e) $(20 - 16x)(3x + 5) - (9 - 6x)(3x - 5) = -45x^2 - 24x + 103$     $\mathbb{G}=\mathbb{R}$

Linke Seite: $60x + 100 - 48x^2 - 80x - (27x - 45 - 18x^2 + 30x) = -48x^2 - 20x + 100 - 27x + 45 + 18x^2 - 30x$

$= -30x^2 - 77x + 145$

Gleichung: $-30x^2 - 77x + 145 = -45x^2 - 24x + 103$

$15x^2 - 53x + 42 = 0$

$x = \frac{53 \pm \sqrt{2809 - 2520}}{30} = \frac{53 \pm 17}{30}$

$x_1 = \frac{70}{30} = \frac{7}{3}$, $x_2 = \frac{36}{30} = \frac{6}{5}$

$\mathbb{L} = \{\frac{6}{5}; \frac{7}{3}\}$

f) $5x^2 + 9x = 0$     $\mathbb{G}=\mathbb{Z}$

$x(5x + 9) = 0$

$x_1 = 0$ oder $x_2 = -\frac{9}{5}$

Da $\mathbb{G}=\mathbb{Z}$ und $-\frac{9}{5} \notin \mathbb{Z}$:

$\mathbb{L} = \{0\}$

g) $7x^2 + 25x - 12 = 0$     $\mathbb{G}=\mathbb{N}$

$x = \frac{-25 \pm \sqrt{625 + 336}}{14} = \frac{-25 \pm \sqrt{961}}{14} = \frac{-25 \pm 31}{14}$

$x_1 = \frac{6}{14} = \frac{3}{7}$, $x_2 = -4$

Da $\mathbb{G}=\mathbb{N}$ und beide Lösungen $\notin \mathbb{N}$:

$\mathbb{L} = \{\}$

h) $4x^2 + 20x + 41 = 0$     $\mathbb{G}=\mathbb{R}$

$x = \frac{-20 \pm \sqrt{400 - 656}}{8} = \frac{-20 \pm \sqrt{-256}}{8}$

Die Diskriminante ist negativ, daher keine reellen Lösungen.

$\mathbb{L} = \{\}$

i) $(x - 5)(x - 6) - (4 - x)(x - 7) = 10$     $\mathbb{G}=\mathbb{Q}$

$x^2 - 11x + 30 - (4x - 28 - x^2 + 7x) = 10$

$x^2 - 11x + 30 - 4x + 28 + x^2 - 7x = 10$

$2x^2 - 22x + 58 = 10$

$2x^2 - 22x + 48 = 0$ bzw. $x^2 - 11x + 24 = 0$

$x = \frac{11 \pm \sqrt{121 - 96}}{2} = \frac{11 \pm 5}{2}$

$x_1 = 8$, $x_2 = 3$

$\mathbb{L} = \{3; 8\}$

j) $(5x - 2)(2 - 5x) - (3x + 2)(x - 4) = 9x + 4$     $\mathbb{G}=\mathbb{Z}$

$(5x - 2)(2 - 5x) = -(5x - 2)^2 = -(25x^2 - 20x + 4) = -25x^2 + 20x - 4$

$(3x + 2)(x - 4) = 3x^2 - 12x + 2x - 8 = 3x^2 - 10x - 8$

$-25x^2 + 20x - 4 - 3x^2 + 10x + 8 = 9x + 4$

$-28x^2 + 30x + 4 = 9x + 4$

$-28x^2 + 21x = 0$

$-7x(4x - 3) = 0$

$x_1 = 0$ oder $x_2 = \frac{3}{4}$

Da $\mathbb{G}=\mathbb{Z}$ und $\frac{3}{4} \notin \mathbb{Z}$:

$\mathbb{L} = \{0\}$
